Git Cheatsheet

2020-07-25 134 words 1 min read times read

Contents

When contributing to opensource projects, git is the default source control tool. I was very used to mercurial(hg) workflow, so it takes me sometime to find out a working source control workflow. Documenting it here for my future self.

set upstream for my local repo

$ git remote add upstream <your_upstream_git_url>

rebase to upstream master

StackOverflow Reference

$ git rebase upstream/master

track remote branch

$ git checkout --track origin/WIP_feature_branch

This will create a local branch named WIP_feature_branch mirros to the remote origin branch.

sync fork to upstream master

This is useful when you see your master is “X commits ahead and Y commits behind” to the upstream master.

git remote add upstream /url/to/upstream/repo
git fetch upstream
git checkout master
git reset --hard upstream/master  
git push origin master --force

hg graft equivalent

git cherry-pick <commit_hash>
